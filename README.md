# Satellite Imagery Based Property Valuation

This repository contains the complete submission for the **Satellite Imagery Based Property Valuation** project.

---

## Repository Structure


---

## Files

- **data_fetcher.py** – Data loading and preparation  
- **preprocessing.ipynb** – Cleaning, EDA, and feature engineering  
- **model_training.ipynb** – Model training, evaluation, and predictions  

- **enrollno_final.csv**  
  - Format: `id,predicted_price`  

- **enrollno_report.pdf**  
  - Final report with EDA and visualizations  

---

## How to Run

1. Run `preprocessing.ipynb`
2. Run `model_training.ipynb`
3. Final predictions are saved as `enrollno_final.csv`
